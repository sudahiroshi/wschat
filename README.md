wschat
======

Example of chat on web browser using groovy and vert.x

## Usage

```
git clone https://github.com/sudahiroshi/wschat.git
cd wschat
vertx run wxchat.groovy
```

Then, you can access ```http://<IP address>:8000/``` from your web browser.
